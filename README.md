# どうぶつずかん

このアプリは、オフラインで見られる動物図鑑です。100種類の動物データと画像をアプリ内に同梱しています。収益化目的ではなく、個人がスマホやPCで見られることを目的にしています。

## PCで開く方法

1. ZIPを展開します。
2. `StartAnimalPictureBook.cmd` をダブルクリックします。
3. ブラウザで `http://127.0.0.1:4174` が開きます。

Node.js が入っていない場合は、先に Node.js をインストールしてください。

`app/index.html` を直接ダブルクリックして開く方法では、画像やオフライン機能が正しく動かない場合があります。必ず `StartAnimalPictureBook.cmd` から起動してください。

## スマホで見る方法

1. PCとスマホを同じWi-Fiにつなぎます。
2. PCで `StartAnimalPictureBook.cmd` を実行します。
3. WindowsのIPアドレスを調べ、スマホのブラウザで `http://PCのIPアドレス:4174` を開きます。

Windowsのファイアウォールが表示された場合は、同じ家庭内ネットワークでのアクセスを許可してください。

## ホーム画面に追加する方法

- iPhone / iPad: Safariで開き、共有ボタンから「ホーム画面に追加」を選びます。
- Android: Chromeで開き、メニューから「ホーム画面に追加」を選びます。
- PC: 対応ブラウザのインストールまたはアプリ追加の表示を使います。

## オフライン利用

初回表示時に、動物データ、画像、画面UIをブラウザへ保存します。一度読み込んだ後は、ネット接続がない状態でも図鑑を見られます。

## 入っているもの

- 動物100種類の図鑑データ
- 動物100種類の写真系画像
- 写真を大きく表示する詳細ページ
- 一覧、検索、お気に入り
- 前へ / 次へ / ランダム / 横スワイプ
- PWA用マニフェストとサービスワーカー
